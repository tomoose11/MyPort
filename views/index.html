This is the first game I made in unity which is far from complete , but it taught me many things. It is a 2D platform game inspired by Super Mario World. Whilst making this game I learned the fundamentals of how to work with a game engine. It taught me how to script events, how you are able to manipulate game objects in the way you want. I was also able to see how quickly a project can grow and the importance of staying organized. Learning about the physics of the game was very interesting.

This is a simple Shopping List App I created for my mum. I used React for the Front End and created an Express REST api to interact with a mongoDB hosted at Mlab. Making this taught me how to use state in React and how React can rerender individual dom elements when they change. It has also allowed me to get more familiar with Node and how to use NPM to install the various dependencies of the app. I focused more on functionality than styling when making this.
